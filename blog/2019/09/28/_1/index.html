SDKMANを使ってkotlinをインストール

2019年9月27日 · 約2分

SDKMANはJVM言語のインストールができて、バージョンも指定できるツールです。

https://sdkman.io/

SDKMANは以下をサポートしているそう。

  • Mac OSX
  • Linux
  • Cygwin
  • Solaris
  • FreeBSD

SDKMANのインストール​

$ curl -s "https://get.sdkman.io" | bash
$ source "$HOME/.sdkman/bin/sdkman-init.sh"
$ sdk version
SDKMAN 5.7.3+337

kotlinのインストール​

9月末時点の最新バージョン（1.3.50）をインストールします。

$ sdk list kotlin
